# MY-WEALTH

부부와 가족이 같은 숫자를 보며 돈 이야기를 쉽게 시작하도록 돕는 자산 성장 가계부입니다.

## Product Positioning

My Wealth is not trying to be another solo expense tracker. The product direction is:

> A shared wealth operating app for couples and families.

Core value:

- Shared household ledger for living costs, personal spending, assets, debt, and investments.
- Fast manual entry assisted by receipt scanning and repeat-friendly flows.
- Household cash-flow reports that help couples discuss what happened this month.
- Wealth goal tracking that connects daily spending to long-term asset growth.

## Near-Term Focus

1. Make the two-person household flow reliable: invite, join, shared data, logout, and session stability.
2. Make daily transaction entry fast enough for real family use.
3. Stabilize receipt scanning with clear failure states and bounded AI cost.
4. Turn the dashboard into a household conversation surface, not just charts.
5. Validate willingness to pay for Family features before broad feature expansion.
